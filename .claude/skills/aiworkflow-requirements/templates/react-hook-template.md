## 使用方法

### 基本的な使用

1. Hookをインポート
2. コンポーネント内でHookを呼び出し
3. 戻り値から必要なプロパティ・アクションを分割代入
4. UIにバインド

### オプション付き使用

1. Hookをインポート
2. 第2引数にオプションオブジェクトを指定
3. 戻り値を使用
