## 使用方法

1. 認証トークンを取得
2. Authorizationヘッダーにトークンを設定
3. Content-Typeヘッダーを`application/json`に設定
4. リクエストボディをJSON形式で送信
5. レスポンスの`success`フィールドで成功/失敗を判定
