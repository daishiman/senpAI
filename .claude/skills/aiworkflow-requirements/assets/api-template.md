## エンドポイント詳細

### GET /{resource}

**用途**: {一覧取得の詳細説明}

**クエリパラメータ**:

| パラメータ | 型     | 必須 | デフォルト | 説明                |
| ---------- | ------ | ---- | ---------- | ------------------- |
| page       | number | -    | 1          | ページ番号          |
| limit      | number | -    | 20         | 取得件数（最大100） |
| sort       | string | -    | -createdAt | ソート順            |
| filter     | object | -    | -          | フィルタ条件        |

**レスポンス（成功時 200）**:

| フィールド | 型      | 説明             |
| ---------- | ------- | ---------------- |
| success    | boolean | true             |
| data       | array   | リソース配列     |
| pagination | object  | ページネーション |

**レスポンス（エラー時）**:

| ステータス | 説明           |
| ---------- | -------------- |
| 400        | パラメータ不正 |
| 401        | 認証失敗       |
| 500        | サーバーエラー |

**新規作成**:

- メソッド: POST
- パス: `/{resource}`
