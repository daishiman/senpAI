## データフロー

**処理の流れ**:

1. ユーザーがUIで操作を実行
2. Renderer ProcessがIPC経由でMain Processに要求
3. Main Processがサービス層で処理
4. 結果をRendererに返却

**依存関係ルール**:

| 依存元         | 依存先         | 許可 |
| -------------- | -------------- | ---- |
| Presentation   | Application    | ✅   |
| Application    | Domain         | ✅   |
| Domain         | Infrastructure | ❌   |
| Infrastructure | Domain         | ✅   |
