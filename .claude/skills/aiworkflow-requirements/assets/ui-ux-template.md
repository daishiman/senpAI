## 状態管理

### Zustand Store

| State      | 型      | 初期値 | 説明             |
| ---------- | ------- | ------ | ---------------- |
| items      | Item[]  | []     | アイテム一覧     |
| selectedId | string  | null   | 選択中ID         |
| isLoading  | boolean | false  | 読み込み中       |
| error      | string  | null   | エラーメッセージ |

### Actions

| Action     | 引数            | 説明         |
| ---------- | --------------- | ------------ |
| setItems   | (items: Item[]) | 一覧設定     |
| selectItem | (id: string)    | アイテム選択 |
| clearError | ()              | エラークリア |

## アクセシビリティ

### WCAG 2.1 AA準拠

| 項目               | 要件                         | 実装方法            |
| ------------------ | ---------------------------- | ------------------- |
| キーボード操作     | 全機能がキーボードで操作可能 | tabIndex, onKeyDown |
| スクリーンリーダー | 適切なラベル付け             | aria-label, role    |
| コントラスト       | 4.5:1以上                    | Design Token使用    |
| フォーカス表示     | 明確なフォーカスインジケータ | outline設定         |

### ARIAパターン

| パターン | 用途           | 主要属性            |
| -------- | -------------- | ------------------- |
| dialog   | モーダル       | role, aria-modal    |
| listbox  | リスト選択     | role, aria-selected |
| menu     | ドロップダウン | role, aria-expanded |
