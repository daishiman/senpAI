| ステージ | トリガー   | 内容           |
| -------- | ---------- | -------------- |
| Build    | Push       | ビルド実行     |

### デプロイ手順

1. PRマージでmainブランチ更新
2. GitHub Actionsがトリガー
3. ビルド・テスト実行
4. 環境別デプロイ
5. ヘルスチェック確認
