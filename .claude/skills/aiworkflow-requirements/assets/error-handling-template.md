## エラー型定義

```typescript
// エラーコード定義
export const ERROR_CODES = {
  // システムエラー
  INTERNAL_ERROR: "ERR_XXXX",

  // ビジネスエラー
  NOT_FOUND: "ERR_XXXX",

  // バリデーションエラー
  INVALID_INPUT: "ERR_XXXX",
} as const;

// エラークラス定義
export class XxxError extends Error {
  constructor(
    public readonly code: string,
    message: string,
    public readonly cause?: Error,
  ) {
    super(message);
    this.name = "XxxError";
  }
}
```

## リトライ戦略

| エラー種別   | リトライ可否 | 最大回数 | 間隔 | バックオフ |
| ------------ | ------------ | -------- | ---- | ---------- |
| ネットワーク | 可           | 3        | 1s   | 指数関数   |
| タイムアウト | 可           | 2        | 2s   | 線形       |
| 認証エラー   | 不可         | -        | -    | -          |
