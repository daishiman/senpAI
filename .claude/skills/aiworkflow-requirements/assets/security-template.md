## 対策

### 入力検証

| 入力タイプ | 検証方法                 | 実装場所        |
| ---------- | ------------------------ | --------------- |
| テキスト   | Zodスキーマ              | API/IPC Handler |
| ファイル   | MIME検証、サイズ制限     | File Handler    |
| URL        | プロトコル・ドメイン検証 | URL Parser      |

### 出力エンコード

| 出力先 | エンコード方法           | ライブラリ     |
| ------ | ------------------------ | -------------- |
| HTML   | HTMLエスケープ           | DOMPurify      |
| JSON   | 自動エスケープ           | JSON.stringify |
| SQL    | プリペアドステートメント | Drizzle ORM    |

## 認証・認可

### 認証方式

| 方式          | 用途         | 有効期限           |
| ------------- | ------------ | ------------------ |
| JWT Bearer    | API認証      | 1時間              |
| Refresh Token | トークン更新 | 7日                |
| API Key       | Local Agent  | 無期限（手動更新） |

### 認可レベル

| レベル | 権限           | 対象操作       |
| ------ | -------------- | -------------- |
| Public | なし           | ヘルスチェック |
| User   | 自分のリソース | CRUD操作       |
| Admin  | 全リソース     | 管理操作       |

### 権限チェック

| チェックポイント | 検証内容       | 失敗時の応答     |
| ---------------- | -------------- | ---------------- |
| API Handler      | トークン有効性 | 401 Unauthorized |
| Service Layer    | リソース所有権 | 403 Forbidden    |
| Data Layer       | アクセス権限   | 404 Not Found    |

## データ保護

### 暗号化

| データ種別     | 保存時               | 転送時  |
| -------------- | -------------------- | ------- |
| 認証情報       | AES-256-GCM          | TLS 1.3 |
| ユーザーデータ | -                    | TLS 1.3 |
| APIキー        | Electron safeStorage | TLS 1.3 |

### 機密データ管理

| データ種別 | 保存場所                  | アクセス制御      |
| ---------- | ------------------------- | ----------------- |
| APIキー    | Electron safeStorage      | Main Process Only |
| トークン   | メモリ（HttpOnly Cookie） | 自動管理          |
| パスワード | 保存しない                | Supabase委譲      |

## Electron固有

### IPC セキュリティ

| 設定項目         | 設定値 | 理由             |
| ---------------- | ------ | ---------------- |
| nodeIntegration  | false  | XSS対策          |
| contextIsolation | true   | コンテキスト分離 |
| sandbox          | true   | プロセス分離     |

### Preload スクリプト

| 公開API            | 許可操作    | 制限事項       |
| ------------------ | ----------- | -------------- |
| window.{domain}API | IPC通信のみ | fsアクセス不可 |
